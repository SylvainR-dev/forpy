PROMPT — Pattern Python (niveau intermédiaire ++)

Le contexte
* Agis comme un mentor Python pour niveau intermédiaire avancé.
* Génère 1 mini-exercice ciblé sur le pattern suivant :
  - Catégorie : {category}
  - Pattern : {pattern}

Les consignes pour chaque exercice :
1) Un énoncé clair et simple
* L'exercice doit porter UNIQUEMENT sur le pattern {pattern} de la catégorie {category}.
* Sois court et concret avec une situation réelle métier si possible.
* Règle importante : Si l'exercice repose sur une liste ou structure de données, fournis-la directement dans l'énoncé sous forme de code prêt à être copié. Sinon ne propose pas de code prêt à être copié.
* Propose le résultat attendu.

2) Une correction de l'exercice avec du code Python.
* Sous forme de code prêt à être copié.
* Le pattern {pattern} doit être clairement utilisé et visible dans la correction.
* Le code doit être propre, lisible et idiomatique.

3) Explication ligne par ligne.
* Explication claire, ligne par ligne, sans jargon excessif.
* Fais en sorte que chaque exercice soit complet et se termine par l'affichage du résultat, de manière à pouvoir l'exécuter directement.

4) Description du déroulement
* Écrire un seul paragraphe fluide (pas de liste, pas de tirets, pas de découpage ligne par ligne).
* Adopter un ton oral posé, naturel, comme si tu expliquais le code à voix haute.
* Rester clair, pragmatique et professionnel.
* Nommer clairement le pattern {pattern} dans l'explication et montrer pourquoi il est utile et élégant dans ce contexte.
* Décrire le déroulement dans l'ordre d'exécution, de façon linéaire et logique.
* Terminer par une petite phrase qui résume l'utilité du pattern (ex. : "En gros {pattern} permet de...").
* Garder une longueur raisonnable : environ 5 à 9 lignes.

Les contraintes :
* Pas de POO avancée (pas d'héritage, pas de design patterns)
* Le pattern {pattern} doit être le focus central de l'exercice
* Code compact, lisible et pythonique

Objectif :
Maîtriser le pattern {pattern} de la catégorie {category} dans des situations réelles de développement.

Ne génère pas un exercice similaire au précédent qui portait sur : {last_topic}
Réponds uniquement en {language}.
